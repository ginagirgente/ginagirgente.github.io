Basemaps

Pirate Watercolor
This was created in ArcGIS Online (Map Viewer) and was inpsired by John Nelson's water color map tutorial. I love how bright John's is, so I decided to create a darker version, which then turned into a pirate-themed map.
Because this was created in Map Viewer, there is not yet a way to directly embed the map into my website, at least that I know of. The feature is available in Map Viewer Classic, but some styling features I used are not supported by Map Viewer Classic. So, unil Map Viewer is able to be embedded, below is a video of my map. You can also access the map here. The ship tracks and symbols are layers (aka not part of the basemap), so you can turn them off or add to them if you want more tracks on the map.
